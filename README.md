# 🌤️ Telegram-бот прогнозу погоди з логуванням

Проєкт являє собою Telegram-бота на базі бібліотеки `python-telegram-bot`, який інтегрується із сервісом погоди OpenWeather API. Бот дозволяє користувачам дізнаватися актуальну погоду в будь-якому місті, автоматично веде облік усіх дій у лог-файл та розраховує персональний час для нагадування про прийом води.

---

## 🛠️ Використовувані API та технології

У проєкті задіяні такі зовнішні інтерфейси та бібліотеки:

1. **[Telegram Bot API](https://telegram.org)** — використовується для керування ботом, відправки текстових повідомлень, обробки користувацьких команд та повідомлень.
2. **[OpenWeather Current Weather Data API](https://openweathermap.org)** — використовується для отримання актуальних метеорологічних даних (температура, вологість, швидкість вітру, опис погоди) за назвою міста.
3. **`python-dotenv`** — бібліотека для безпечного читання конфігураційних даних та секретних ключів з ізольованого файлу `.env`.
4. **`requests`** — бібліотека для відправки синхронних HTTP-запитів до серверів погоди.
5. **`logging`** (вбудований модуль Python) — інструмент для фіксації історії роботи програми, успішних запитів та відладки помилок.

---

## 🚀 Як запустити проєкт?

### 1. Підготовка конфігурації
Створіть у кореневій папці проєкту файл із назвою `.env` та додайте ваші секретні токени:

```env
API_KEY=ваш_фактичний_api_ключ_від_openweather
TELEGRAM_BOT_TOKEN=ваш_токен_отриманий_від_botfather
```

### 2. Встановлення залежностей
Встановіть необхідні пакети через менеджер пакетів `pip`:

```bash
pip install python-telegram-bot requests python-dotenv
```

### 3. Запуск бота
Запустіть основний скрипт програми з терміналу:

```bash
python bot.py
```

Після успішного запуску в терміналі з'явиться підтверджуючий запис про початок роботи.

## 📸 Приклади роботи програм

### 💻 Відображення в інтерфейсі Telegram
Користувач взаємодіє з ботом за допомогою вбудованих команд та звичайного тексту:

> **Користувач:** `/start`  
> **Бот:** 👋 Привіт, Іван! Радий тебе бачити. Я погодний бот. Використовуй команду /help, щоб побачити список доступних команд.
>
> **Користувач:** `/weather`  
> **Бот:** 🌍 Будь ласка, введіть назву міста (наприклад, Одеса):  
> **Користувач:** `Одеса`  
> **Бот:**  
> 🌡️ **Погода в місті Одеса:**  
> • Температура: 24.5°C  
> • Стан: Ясно  
> • Вологість: 60%  
> • Швидкість вітру: 3.4 м/с  
>   
> 🔔 *Нагадування від бота:* Не забудьте випити склянку води о **17:52**!

### 📝 Записи у файлі логів `app.log`
Кожна дія користувача, мережевий запит та нагадування фіксуються у файлі `app.log` в режимі реального часу:

```text
2026-06-27 16:50:00 [INFO] Бот успішно запущений і почав опитування серверів Telegram...
2026-06-27 16:51:12 [INFO] Користувач Іван (ID: 12345678) натиснув /start
2026-06-27 16:51:20 [INFO] Користувач Іван (ID: 12345678) ініціював команду /weather
2026-06-27 16:52:05 [INFO] Користувач Іван (ID: 12345678) запросив погоду для міста: Одеса
2026-06-27 16:52:06 [INFO] Успішно відправлено погодні дані для міста Одеса користувачеві Іван
2026-06-27 16:52:06 [INFO] ⏰ Нагадування записано в лог: час випити води — 17:52
```

У разі введення неіснуючого міста або збою авторизації в лог заноситься запис зі статусом `[ERROR]`.
